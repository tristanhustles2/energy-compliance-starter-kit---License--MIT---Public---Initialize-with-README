# NERC CIP Gap Analysis Checklist
**Infrastructure Velocity | Day 1 Discovery**

## Pre-Review
- [ ] Audit scope / notification letter received
- [ ] Existing NERC CIP policies received
- [ ] Network diagrams received
- [ ] Asset inventory (BES Cyber Systems) received
- [ ] Previous audit findings received

## CIP-002 — Asset Categorization
- [ ] Current BES Cyber System categorization approved?
- [ ] CIP-002-8 reclassification performed for low-impact assets?
- [ ] New DER/inverter assets inventoried and categorized?

## CIP-005 — Electronic Security Perimeters
- [ ] ESP documented with current network diagram?
- [ ] DER communication channels included in ESP scope?
**Common Gap:** New inverter communication paths not in ESP boundary.

## CIP-007 — Systems Security Management
- [ ] Ports and services documented and reviewed?
- [ ] Patch management current (35-day cycle)?
- [ ] User access reviewed in last 15 months?
**Common Gap:** Patch gaps on OT systems — vendors delay patches.

## CIP-010 — Configuration Change Management
- [ ] Baseline configuration documented for each BES Cyber System?
- [ ] All changes documented and approved before implementation?
**Common Gap:** Undocumented changes during emergency maintenance.

## CIP-013 — Supply Chain Risk
- [ ] Supply chain risk management plan documented?
- [ ] Inverter and DER vendors included in supply chain review?
**Common Gap:** Clean energy vendors not yet in supply chain risk process.
